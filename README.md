# Project-Woman-Basic
This project focuses on enhancing the user experience of the Woman Basic fashion shop through improvements in system operations, performance optimization, and the integration of AI technologies, thereby providing a more efficient and engaging experience for both the business and its customers.

## Getting Started

This project uses **npm workspaces** to manage both the frontend and backend from a single root directory.

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) (v7 or higher)

### Installation
Run the following command at the root of the project to install all dependencies for both frontend and backend:
```bash
npm install
```

### Running the Project
To start both the **Frontend** and **Backend** simultaneously in development mode:
```bash
npm run dev
```

### Other Commands
- `npm run frontend`: Starts only the frontend application.
- `npm run backend`: Starts only the backend server.
- `npm run install-all`: Re-installs dependencies for all folders.
